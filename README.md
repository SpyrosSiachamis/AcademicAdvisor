# Academic Advisor

An academic decision-support system that recommends courses and academic paths based on completed courses, ratings, workload tolerance, interests, prerequisites, degree constraints, and career goals.


## Current Status

Early-stage development. The project currently contains a containerized FastAPI backend used for experimenting with API structure, validation, and backend architecture before implementing the main recommendation system.

## Planned Stack

- FastAPI
- PostgreSQL
- Redis
- Docker
- Next.js
- Agentic tooling layer
